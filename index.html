Correlate-sleep

This is an app that correlates every event in your life with your sleep performance, so you can analyze and find root causes of why your sleep performance is like it is.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Correlate Sleep, a member of the Correlate Family.

This Correlate module helps you understand your sleep patterns, beyond what the usual app of the tracker can provide. As you start linking new services, you'll start discovering behaviors and patterns that will give you clues on how to improve (or, at least, understand) your sleep time.
